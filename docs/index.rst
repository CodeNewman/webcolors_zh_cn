webcolors
=========

This module provides utility functions for working with the color
names and color value formats defined by the HTML and CSS
specifications for use in documents on the Web.

For example:

.. code-block:: pycon

    >>> import webcolors
    >>> webcolors.hex_to_name('#daa520')
    'goldenrod'
